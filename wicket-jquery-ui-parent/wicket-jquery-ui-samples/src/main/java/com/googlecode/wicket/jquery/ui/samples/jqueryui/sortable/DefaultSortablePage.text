The supplied list should be modifiable for the new order to be reflected on 'update' event. That means, the use of <code>Arrays#asList()</code> or <code>Collections#unmodifiableList()</code> should be avoided.<br/>
If you don't want the new order is reflected to the model object, remove the call to <code>super.onUpdate(...)</code>
