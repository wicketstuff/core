The Wizard component works like the built-in <code>Wizard</code> component; It uses an <code>IWizardModel</code> (<code>WizardModel</code> or <code>DynamicWizardModel</code>) which may contains several <code>WizardStep(s)</code> or <code>DynamicWizardStep(s)</code><br/> 
<br/>
Wizard button's text can be customized using the following resource keys, either in <i>MyApplication</i>.properties, <i>MyPage</i>.properties or <i>MyWizard</i>.properties:
<ul>
	<li><code>wizard.button.prev=My previous</code></li>
	<li><code>wizard.button.next=My next</code></li>
	<li><code>wizard.button.last=My last</code></li>
	<li><code>wizard.button.finish=My finish</code></li>
	<li><code>wizard.button.cancel=My cancel</code></li>
</ul>
