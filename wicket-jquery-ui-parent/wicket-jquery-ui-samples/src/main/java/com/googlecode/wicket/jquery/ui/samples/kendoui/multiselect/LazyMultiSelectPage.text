This sample shows how to use the Kendo UI <tt>MultiSelect</tt> with lazy loading choices.

<br/>
<h4>Note</h4>
This <tt>MultiSelect</tt> implementation - in the "lazy" package - differs from the other one as it does not inherit from <tt>ListMultipleChoice</tt> built-in component.

<br/>
<h4>Warning</h4>
Do not re-attach the <tt>MultiSelect</tt> directly to the <tt>AjaxRequestTarget</tt>, attach its container instead (the form for instance)
