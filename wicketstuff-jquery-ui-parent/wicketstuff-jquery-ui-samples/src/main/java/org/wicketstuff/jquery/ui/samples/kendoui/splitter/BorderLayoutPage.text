This sample shows how to use the <code>BorderLayout</code> based on Kendo UI <code>SplitterBehavior(s)</code>. Look at the HTML source to see how to set the layout.<br/>
<br/>
You may override <code>getVerticalPanes()</code> and <code>getHorizontalPanes()</code>, which return panes definitions in a JSON format.<br/>

<h4>Note:</h4>
The <code>BorderLayout</code> IS-A <code>WebMarkupContainer</code>. If you wish to apply a <code>BorderLayout</code> on an existing <code>Page</code> or an existing <code>Panel</code>, you may implement the <code>IBorderLayout</code> interface.
