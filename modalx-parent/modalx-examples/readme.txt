If you have maven installed:
----------------------------

To build and run this example app using maven:

mvn -Djetty.port=80 jetty:run


Open a browser and type in:

http://localhost/modalx

If you don't have maven installed:
----------------------------------

There is a prebuilt modalx.war file inside the example/dist directory that 
can simply be simply copied to the webapps directory of a servlet 
container (eg., <tomcat>/webapps) to install.
