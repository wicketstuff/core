This distribution contains two directories:

libdev: 
	The ModalX framework source and binaries used to build the ModalX library (built with ant)
	See the readme.txt file under 'libdev' for more information

example: 
	An example application that demonstrates some of the key features of ModalX (depends on the
	jar located in libdev/dist)
	See the readme.txt file under 'example' for instructions on how to run the example.


It also contains a gettingstarted.txt file that will guide you through the basic concepts.
