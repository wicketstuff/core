ObjectAutoComplete

wicketstuff-objectautocomplete is a component for identifying an object from an autocomplete suggestion list in a 'save' way (ie. without the semantic issues discussed here)

Examples

  • Simple Example - Simple Example showing the default behaviour
  • Listener Example - Example the usage of a listener which gets called on each input change
  • ReadOnlyObjectRenderer Example - how to use a custom panel for showing the selected's object identity
  • Validation Example - how to make an objectautocomplete field required
  • Complex Choices Example - how to provide a more than a simple list as choices
  • Delay Example - how to delay the server lookup
