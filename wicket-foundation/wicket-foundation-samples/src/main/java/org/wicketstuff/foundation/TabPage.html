Tabs

Tabs are elements that help you organize and navigate multiple documents in a single container. They can be used for switching between items in the container.

Horizontal Tabs

You can create horizontal tabs using minimal markup.

HTML

<div wicket:id="tabHorizontal"></div>

Java

add(new FoundationButton("btn", Model.of("Click")));

Rendered HTML

Vertical Tabs

You can create vertical tabs using minimal markup.

HTML

<div wicket:id="tabVertical"></div>

Java

add(new FoundationTab("tabVertical", tabs).setVerticalTab(true));

Rendered HTML

Ajax-enhanced Tabs

You can create Ajax-enhanced tabs using minimal markup.

HTML

<div wicket:id="ajaxTab"></div>

Java

add(new AjaxFoundationTab("ajaxTab", tabs));
