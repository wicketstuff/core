Below is a map covered with WFS data, each marker represents a historic site in Massachusetts. As you pan and zoom the map, more data will be loaded on demand. If you click on a point, more information about the feature will appear in a popover window. The red marker indicates the location of Miles' office.

Beware: If you zoom out to far, you may be in for a lengthy wait as the map waits for all of the points from the data source to load.

The data is provided by the Massachusetts Historial Commisssion.
