Below is a map covered with clustered data, each cluster displays the number of historical sites present inside the cluster and is drawn proportional to the number of sites.

The clustering is done entirely by the map. All of the points in the viewable extent are loaded from the data source and the clustering is performed client side. As you pan and zoom, all of the clusters are re-calculated.

The data is provided by the Massachusetts Historial Commisssion.

Beware

  • If you zoom out sufficiently far, you will have to wait (perhaps quite a while) for the clusters to be calculated.
