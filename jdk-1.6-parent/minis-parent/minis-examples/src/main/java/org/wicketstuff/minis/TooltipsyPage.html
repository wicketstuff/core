wicket-tooltipsy

tooltipsy is a jQuery tooltip plugin created by Brian Cray because he was frustrated with the lack of CSS control and bulky includes of the others.

wicket-tooltipsy is Apache Wicket integration to tooltipsy created by Ilkka Seppälä.
